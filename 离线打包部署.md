# 离线部署指南

## 文件清单

在离线环境部署前，需要从在线环境打包以下内容：

| 项目 | 大小 | 来源服务器 |
|------|------|-----------|
| 应用镜像 (2 个) | ~200 MB | 10.19.26.148 |
| PaddleOCR-VL 镜像 (2 个) | ~14 GB | 10.19.26.153 |
| MinerU 镜像 (1 个) | ~18 GB | 10.19.26.153 |
| 应用代码 + Docker Compose | ~5 MB | 本项目目录 |

---

## 第一步：打包（在在线环境执行）

### 打包应用

```bash
cd /opt/pdf2searchable-v2

# 1. 打包 Docker 镜像
docker save pdf2searchable-v2-backend:latest pdf2searchable-v2-nginx:latest | gzip > images/app_images.tar.gz

# 2. 打包代码（不含 node_modules、__pycache__ 等）
tar czf app_code.tar.gz \
  --exclude='frontend/node_modules' \
  --exclude='backend/__pycache__' \
  --exclude='backend/app/__pycache__' \
  --exclude='backend/app/formatters/__pycache__' \
  --exclude='.git' \
  --exclude='frontend/dist' \
  -C /opt/pdf2searchable-v2 .
```

### 打包算力服务器镜像

SSH 到算力服务器执行：

```bash
ssh root@10.19.26.153

# PaddleOCR-VL 镜像（较大的两个）
docker save \
  ccr-2vdh3abv-pub.cnc.bj.baidubce.com/paddlepaddle/paddleocr-vl:latest-nvidia-gpu-offline \
  ccr-2vdh3abv-pub.cnc.bj.baidubce.com/paddlepaddle/paddleocr-genai-vllm-server:latest-nvidia-gpu-offline \
  | gzip > /opt/paddleocr_package/paddleocr_images.tar.gz

# MinerU 镜像
docker save mineru:latest | gzip > /opt/paddleocr_package/mineru_image.tar.gz

# 配置文件
tar czf /opt/paddleocr_package/configs.tar.gz \
  -C /opt/paddleocr-vl compose.yaml pipeline_config_vllm.yaml
```

### 传输到移动介质

```bash
# 把所有包传到一块移动硬盘或U盘
# 总量约 32GB，建议用 NTFS/exFAT 格式的硬盘
cp images/app_images.tar.gz /mnt/usb/
cp app_code.tar.gz /mnt/usb/
# 从算力服务器 scp 过来再拷贝
scp root@10.19.26.153:/opt/paddleocr_package/*.tar.gz /mnt/usb/
scp root@10.19.26.153:/opt/paddleocr_package/configs.tar.gz /mnt/usb/
```

---

## 第二步：部署（在离线环境执行）

### 1. 加载镜像

```bash
# 应用服务器
gunzip -c images/app_images.tar.gz | docker load

# 算力服务器
gunzip -c images/paddleocr_images.tar.gz | docker load
gunzip -c images/mineru_image.tar.gz | docker load
```

### 2. 解压代码

```bash
mkdir -p /opt/pdf2searchable-v2
tar xzf app_code.tar.gz -C /opt/pdf2searchable-v2
cd /opt/pdf2searchable-v2
```

### 3. 配置算力服务

```bash
mkdir -p /opt/paddleocr-vl
tar xzf configs.tar.gz -C /opt/paddleocr-vl

# 启动 PaddleOCR-VL
cd /opt/paddleocr-vl && docker compose up -d

# 启动 MinerU（假设镜像已导入为新名称，按实际情况调整）
docker run -d --gpus all --name mineru-api ...  # 按原来启动参数
```

### 4. 修改环境变量

编辑 `/opt/pdf2searchable-v2/.env`，把 IP 改成新环境的：

```bash
# 原来
PADDLEOCR_URL=http://10.19.26.153:8080
MINERU_URL=http://10.19.26.153:8000

# 改成新算力服务器的 IP
PADDLEOCR_URL=http://新算力IP:8080
MINERU_URL=http://新算力IP:8000
```

### 5. 启动应用

```bash
cd /opt/pdf2searchable-v2
docker compose up -d
# 访问 http://新应用IP:9010
```

---

## 附：启动脚本

创建 `/opt/pdf2searchable-v2/start.sh`，一键启动所有服务：

```bash
#!/bin/bash
# 应用服务器：启动 PDF2Searchable
cd /opt/pdf2searchable-v2 && docker compose up -d
echo "✅ 应用服务已启动: http://$(hostname -I | awk '{print $1}'):9010"
```

创建 `/opt/paddleocr-vl/start.sh`，一键启动 OCR 服务：

```bash
#!/bin/bash
# 算力服务器：启动 OCR 服务
cd /opt/paddleocr-vl && docker compose up -d
# MinerU 按实际部署命令启动
echo "✅ OCR 服务已启动"
```
